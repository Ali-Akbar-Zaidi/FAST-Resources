The images in this directory on the CD-ROM (ASFD/apache/ref/images)
are used by the HTML 3.2 recommendation document, which is also
on the CD-ROM as ASFD/apache/ref/REC-html32.html.  If you use
your browser to access the recommendation document on the CD-ROM,
the pictures will appear in the browser window as intended.

If you're using a browser on Windows, use the following URL (change
the D drive letter to whatever your CD drive letter is):

    file:///D|/ASFD/apache/ref/REC-html32.html

If you're using a browser on UNIX, use the following URL (assuming
your CD-ROM is mounted on /mnt):

    file:/mnt/ASFD/apache/ref/REC-html32.html
